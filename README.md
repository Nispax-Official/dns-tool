DNS Tool

A simple DNS lookup tool built with Next.js.
It lets you check common DNS records (A, AAAA, MX, TXT, NS, etc.) in a clean interface without any shady stuff or heavy setup.

What it does:
DNS record lookup using DNS over HTTPS
Clean, readable results
Dark UI with animated background
Optional static images / GIFs (no tracking, no scripts)

Tech:
- Next.js
- React
- TypeScript

Run locally:
> npm install
> npm run dev

Open http://localhost:3000

Notes:
This is a legit DNS utility, not a hacking tool.
